# 模組 2：客戶透視鏡（NotebookLM 版）

> **「見面之前，你就已經贏了」**

使用 NotebookLM 快速分析客戶資料，產出情報卡、痛點對照表、會議速查。

---

## 課前準備（學員需上傳的資料）

### 1. 自己公司資料

| 資料類型 | 內容 | 來源 |
|---------|------|------|
| 產品說明 | 服務介紹、功能特色、定價 | 官網 / 簡報 PDF |
| 成功案例 | 過去客戶的合作故事 | 案例報告 / 新聞稿 |
| 競爭優勢 | 跟競品的差異化 | 內部文件 |

### 2. 目標客戶資料

| 資料類型 | 內容 | 來源 |
|---------|------|------|
| 公司官網 | 關於我們、產品服務 | 貼上網址 |
| 年報/財報 | 營運數據、策略方向 | PDF 上傳 |
| 新聞稿 | 近期動態、重大消息 | 貼上網址 |
| 產業報告 | 市場趨勢、產業挑戰 | PDF 上傳 |

### 3. 決策者資料

| 資料類型 | 內容 | 來源 |
|---------|------|------|
| LinkedIn | 個人經歷、學歷、技能 | 貼上網址或截圖 |
| 演講/文章 | 公開發言、專業觀點 | 貼上網址 |
| 社群媒體 | 關注議題、興趣 | 截圖上傳 |

---

## NotebookLM 設定步驟

1. 前往 [NotebookLM](https://notebooklm.google.com/)
2. 點擊「新增筆記本」
3. 上傳/貼上上述所有資料
4. 等待 AI 處理完成
5. 依序貼入以下 3 個 Prompt

---

## Prompt 2：產品 × 客戶痛點對照表

複製以下內容，貼到 NotebookLM 對話框：

```
請根據我上傳的「我司產品資料」和「客戶公司資料」，幫我製作一份「產品 × 客戶痛點對照表」。

分析步驟：
1. 從客戶資料中，找出他們可能面臨的 3-5 個業務痛點/挑戰
2. 從我司產品資料中，找出對應的解決方案
3. 如果有相關成功案例，一併列出

請用以下表格格式呈現：

【產品 × 客戶痛點對照表】

| 客戶痛點 | 痛點說明 | 我們的解決方案 | 帶來的效益 | 相關案例 |
|---------|---------|---------------|-----------|---------|
| 痛點 1 | | | | |
| 痛點 2 | | | | |
| 痛點 3 | | | | |
| 痛點 4 | | | | |
| 痛點 5 | | | | |

表格後，請補充：

■ 最強切入點
根據分析，建議從「___」這個痛點切入，因為：
1.
2.

■ 話術建議
「我們發現很多像貴公司這樣的___企業，都面臨___的挑戰。我們過去幫助___公司，透過___，達成了___的成果。不知道貴公司目前在這方面的情況如何？」
```

## 進階 Prompts（選用）

### 追問 1：競爭對手分析

```
根據這個客戶的產業和需求，他們可能也在考慮哪些競爭對手？
請分析：
1. 可能的競品有哪些？
2. 我們跟這些競品比，優勢在哪？
3. 如果客戶提到競品，我該怎麼回應？
```

### 追問 2：決策者個性分析

```
根據決策者的 LinkedIn 和公開資料，分析：
1. 他可能的溝通風格？（結果導向/關係導向/細節導向/創新導向）
2. 跟他開會時，我應該注意什麼？
3. 什麼樣的話會讓他有感？什麼話要避免？
```

### 追問 3：模擬對話練習

```
請扮演這位決策者，用他可能的口吻和我進行一場模擬會議。
你會：
1. 問一些尖銳的問題
2. 提出常見的異議（價格、時間、風險）
3. 讓我練習回應

準備好了就開始，你先開場。
```
