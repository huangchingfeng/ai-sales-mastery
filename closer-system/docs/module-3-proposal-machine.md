# 模組 3：提案印鈔機（Manus 版）

> **「3 分鐘，一份客製化提案」**

使用 Manus AI 代理，自動搜尋客戶資料、結合你的產品優勢，產出完整提案文件。

---

## 為什麼用 Manus？

| 功能 | Manus | 一般 AI |
|------|-------|--------|
| 自動搜尋網路 | ✅ | ❌ |
| 產出 Word/PDF | ✅ | ❌ |
| 多步驟自動執行 | ✅ | ❌ |
| 結合多來源資料 | ✅ | ❌ |

Manus 可以：
1. 搜尋客戶公司官網、新聞、財報
2. 分析客戶可能的需求和痛點
3. 結合你的產品資料，產出客製化提案
4. 直接輸出成 Word 或 PDF 格式

## Manus Prompt：提案產生器

### Prompt 1：完整提案（一鍵產出）

複製以下內容，貼到 Manus：

```
請幫我製作一份給「[客戶公司名稱]」的業務提案書。

## 步驟

1. **搜尋客戶資料**
   - 搜尋「[客戶公司名稱]」的官網、最新新聞、公司介紹
   - 了解他們的產業、規模、主要業務
   - 找出他們可能面臨的挑戰或需求

2. **分析需求**
   - 根據搜尋結果，推測他們可能的痛點
   - 思考我的產品/服務如何解決這些問題

3. **製作提案書**
   - 用以下結構產出提案：

---

# [客戶公司名稱] 合作提案書

## 1. 關於貴公司（展現我有做功課）
- 簡述對方公司背景
- 提到近期動態或成績
- 點出可能的挑戰/機會

## 2. 我們觀察到的機會
- 列出 2-3 個我們可以幫忙的地方
- 連結到對方的業務目標

## 3. 建議方案
- 方案概述
- 具體服務內容
- 預期效益

## 4. 成功案例
- 相似產業/規模的客戶
- 他們的問題
- 我們的解決方案
- 達成的成果

## 5. 投資與回報
- 費用說明
- ROI 分析（如適用）

## 6. 下一步
- 建議的行動
- 聯絡方式

---

## 我的公司資料（請用這些資料）

[在這裡貼上你準備好的「提案素材包」]

## 輸出格式

請產出 Word 文件格式（.docx），包含：
- 專業的排版
- 清晰的標題層級
- 適當的間距
```

