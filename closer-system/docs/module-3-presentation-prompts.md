# 模組 3：提案印鈔機 - 簡報提示詞庫

> **「3 分鐘，一份客製化簡報」**

使用 Manus + NotebookLM，快速產出 10 種業務簡報。

---

## 工具分工

| 工具 | 適合用於 | 優勢 |
|------|---------|------|
| **Manus** | 需要搜尋外部資料的簡報 | 自動搜尋網路、產出文件 |
| **NotebookLM** | 基於自己資料的簡報 | 結合上傳的文件、案例庫 |

---

## 10 種簡報提示詞

### 📊 類型 1：客製化提案簡報

**使用時機**：針對特定客戶的提案
**建議工具**：Manus（可搜尋客戶資料）

#### Manus Prompt

```
請幫我製作一份給「[客戶公司名稱]」的提案簡報。

## 步驟
1. 搜尋「[客戶公司名稱]」的官網、新聞、公司背景
2. 分析他們可能的需求和痛點
3. 製作 10 頁簡報

## 簡報結構

第 1 頁：封面
- 標題：[客戶公司名稱] 合作提案
- 副標題：[我的公司名稱]
- 日期

第 2 頁：關於貴公司
- 展現我有做功課
- 簡述對方公司亮點
- 提到近期動態

第 3 頁：我們觀察到的機會
- 列出 2-3 個可以幫忙的地方
- 連結到對方的業務目標

第 4 頁：如果不處理...
- 維持現狀的代價
- 競爭對手可能在做什麼

第 5 頁：解決方案概述
- 一句話說明我們怎麼幫
- 核心價值主張

第 6 頁：具體服務內容
- 列點說明
- 包含什麼、不包含什麼

第 7 頁：成功案例
- 相似產業/規模的客戶
- 問題 → 解決方案 → 成果

第 8 頁：投資與回報
- 費用說明
- ROI 分析

第 9 頁：為什麼選我們
- 3 個差異化優勢

第 10 頁：下一步
- 建議的行動
- 聯絡方式

## 我的公司資料

公司名稱：[填入]
主要產品/服務：[填入]
產品優勢：
1. [填入]
2. [填入]
3. [填入]

成功案例：
- 客戶：[填入]，成果：[填入]

## 輸出格式

請產出：
1. 每頁的標題和重點內容
2. 每頁的講者備註（要講什麼、講多久）
3. 視覺建議（適合放什麼圖表/圖片）
```

### 📦 類型 2：產品介紹簡報

**使用時機**：向新客戶介紹產品/服務
**建議工具**：NotebookLM（基於自己的產品資料）

#### NotebookLM Prompt

```
請根據我上傳的產品資料，製作一份「產品介紹簡報」大綱。

## 簡報結構（8 頁）

第 1 頁：封面
- 產品名稱
- 一句話價值主張

第 2 頁：你是否遇到這些問題？
- 列出 3 個目標客戶的常見痛點
- 讓聽眾產生共鳴

第 3 頁：傳統解決方案的問題
- 現有方法的限制
- 為什麼需要新的解決方案

第 4 頁：介紹我們的產品
- 產品是什麼
- 一句話說明核心功能

第 5 頁：產品特色
- 3-5 個主要功能/優勢
- 每個功能對應解決什麼問題

第 6 頁：使用方式
- 怎麼開始使用
- 多快能看到效果

第 7 頁：客戶見證
- 1-2 個成功案例
- 具體數據或引言

第 8 頁：下一步
- 如何購買/試用
- 聯絡方式

## 輸出要求

每一頁請提供：
1. 標題
2. 3-5 個重點內容
3. 視覺建議（適合放什麼）
4. 講者備註（怎麼講、講多久）
```

### ⚖️ 類型 4：方案比較簡報

**使用時機**：比較不同方案/套餐
**建議工具**：NotebookLM

#### NotebookLM Prompt

```
請根據我上傳的產品資料，製作一份「方案比較簡報」大綱。

## 簡報結構（6 頁）

第 1 頁：封面
- 標題：找到最適合你的方案
- 副標題：[產品名稱] 方案總覽

第 2 頁：你的需求是什麼？
- 不同客戶的不同需求
- 引導思考「我需要什麼」

第 3 頁：方案總覽
- 表格比較各方案
- 功能、價格、適合對象

第 4 頁：方案 A 詳細說明
- 適合誰
- 包含什麼
- 價格

第 5 頁：方案 B 詳細說明
- 適合誰
- 包含什麼
- 價格

第 6 頁：如何選擇 + 下一步
- 選擇建議
- 聯絡方式

## 輸出要求

重點呈現：
1. 清晰的比較表格
2. 每個方案的「適合對象」
3. 價格 vs 價值的說明
```

### 💰 類型 5：報價說明簡報

**使用時機**：解釋報價內容和價值
**建議工具**：NotebookLM

#### NotebookLM Prompt

```
請根據我上傳的報價資料，製作一份「報價說明簡報」大綱。

## 簡報結構（6 頁）

第 1 頁：封面
- 標題：[客戶名稱] 專案報價
- 日期

第 2 頁：專案範圍確認
- 客戶的需求是什麼
- 我們要做什麼

第 3 頁：服務內容明細
- 包含什麼（列點）
- 不包含什麼（避免誤解）

第 4 頁：報價明細
- 費用拆解
- 為什麼這樣定價

第 5 頁：投資回報
- 這個投資能帶來什麼
- ROI 試算（如適用）

第 6 頁：下一步
- 報價有效期
- 如何確認
- 付款方式

## 輸出要求

1. 清晰的價格說明
2. 強調「價值」而非「價格」
3. 降低客戶疑慮的說明
```

### 🏆 類型 6：成功案例簡報

**使用時機**：展示過去的成功故事
**建議工具**：NotebookLM

#### NotebookLM Prompt

```
請根據我上傳的案例資料，製作一份「成功案例簡報」大綱。

## 簡報結構（每個案例 3-4 頁）

### 案例 1

第 1 頁：案例封面
- 客戶名稱 + Logo
- 產業、規模
- 一句話成果

第 2 頁：挑戰
- 客戶原本的問題是什麼
- 嘗試過什麼方法
- 為什麼找到我們

第 3 頁：解決方案
- 我們做了什麼
- 怎麼做的
- 時間軸

第 4 頁：成果
- 具體數據
- 客戶見證/引言
- 後續發展

### 案例 2、3...（重複結構）

## 輸出要求

1. 每個案例都要有「數據」
2. 包含客戶的真實引言（如有）
3. 突顯與聽眾的相關性
```

### ❓ 類型 7：FAQ 簡報

**使用時機**：預先準備常見問題
**建議工具**：NotebookLM

#### NotebookLM Prompt

```
請根據我上傳的產品資料和常見問題，製作一份「FAQ 簡報」大綱。

## 簡報結構

第 1 頁：封面
- 標題：常見問題解答
- 副標題：你想知道的都在這裡

接下來每頁一個問題：

第 2 頁：Q1 - 價格相關
- 問題：[最常問的價格問題]
- 回答：[簡潔有力的回應]
- 補充：[如果客戶還有疑慮...]

第 3 頁：Q2 - 效果相關
第 4 頁：Q3 - 時間相關
第 5 頁：Q4 - 比較相關
第 6 頁：Q5 - 風險相關

最後一頁：還有問題？
- 聯絡方式
- 預約諮詢

## 請整理出客戶最常問的 5 個問題

類別建議：
1. 價格/費用
2. 效果/成果
3. 時間/流程
4. 與競品比較
5. 風險/保障

每個問題提供：
1. 問題本身
2. 簡短回答（30 秒內講完）
3. 詳細說明（如需要）
4. 佐證資料（案例、數據）
```

### 🚀 類型 9：追加銷售簡報

**使用時機**：推薦現有客戶升級或加購
**建議工具**：NotebookLM

#### NotebookLM Prompt

```
請根據我上傳的客戶資料和產品資料，製作一份「追加銷售簡報」大綱。

## 簡報結構（6 頁）

第 1 頁：封面
- 標題：讓成果更上一層樓
- 副標題：[客戶名稱] 專屬建議

第 2 頁：目前的成果
- 回顧已經達成的成果
- 肯定合作的價值

第 3 頁：我們發現的機會
- 還可以做得更好的地方
- 新的需求/挑戰

第 4 頁：建議方案
- 追加/升級的內容
- 能帶來什麼額外價值

第 5 頁：投資與回報
- 追加費用
- 預期的額外效益

第 6 頁：下一步
- 如何升級
- 特別優惠（如有）

## 輸出要求

1. 從「感謝」和「成果」開始
2. 用「機會」而非「問題」的角度
3. 強調「更好」而非「不夠」
```

### 🤝 類型 10：轉介紹說明簡報

**使用時機**：方便客戶幫你介紹
**建議工具**：NotebookLM

#### NotebookLM Prompt

```
請根據我上傳的公司和產品資料，製作一份「轉介紹說明簡報」大綱。

這是給現有客戶看的，讓他們可以輕鬆把我介紹給朋友。

## 簡報結構（5 頁）

第 1 頁：封面
- 標題：把好東西分享給你的朋友
- 副標題：[我的公司名稱] 簡介

第 2 頁：我們幫誰解決什麼問題
- 目標客戶描述
- 我們解決的核心問題
- 一句話說明

第 3 頁：為什麼客戶選擇我們
- 3 個核心優勢
- 用簡單的話說明

第 4 頁：客戶見證
- 2-3 個成功案例
- 簡短的客戶好評

第 5 頁：如何聯繫
- 聯絡方式
- 轉介紹流程
- 感謝禮（如有）

## 輸出要求

1. 用客戶的語言（不是業務話術）
2. 簡單到客戶可以直接轉發
3. 清楚說明「誰適合」
```

## 課堂流程建議（1.5 小時）

| 時間 | 活動 | 說明 |
|------|------|------|
| 0:00-0:10 | 介紹 | 10 種簡報類型總覽 |
| 0:10-0:20 | 工具說明 | Manus vs NotebookLM 使用時機 |
| 0:20-0:40 | 實作 1 | 學員選擇最需要的簡報類型，產出第一份 |
| 0:40-0:55 | 實作 2 | 產出第二份簡報 |
| 0:55-1:10 | 優化調整 | 修改、補充、客製化 |
| 1:10-1:20 | 分享 | 學員展示成果 |
| 1:20-1:30 | 總結 | Q&A、提示詞庫使用說明 |
