# 模組 4：跟進永動機

> **「再也不漏掉任何一個客戶」**

使用 Gemini Gem 建立專屬的「跟進訊息助理」，6 種情境一鍵產出。

---

## 建立你的跟進 Gem

### 步驟

1. 前往 [Gemini](https://gemini.google.com/)
2. 點擊左側「Gems」→「新增 Gem」
3. 貼上以下系統提示詞
4. 儲存後即可隨時使用

### Gem 系統提示詞

```
你是我的專屬跟進訊息助理，幫我產出各種業務跟進訊息。

## 我的背景資料

姓名：[填入你的名字]
公司：[填入公司名稱]
產品/服務：[填入主要產品]
說話風格：[填入：專業正式 / 親切口語 / 幽默輕鬆]

## 你的任務

當我告訴你：
- 跟進情境（會議後、沉默客戶、節日問候等）
- 客戶資訊（名字、公司、上次互動）
- 發送管道（Email / LINE / WhatsApp）

你就幫我產出合適的跟進訊息。

## 產出原則

1. 用我的說話風格
2. 簡潔有力，不囉嗦
3. 有明確的行動呼籲
4. 讓客戶感受到真誠，不是罐頭訊息

## 輸出格式

每次產出 3 個版本讓我選：
- 版本 A：正式版
- 版本 B：親切版
- 版本 C：簡短版

每個版本標註建議的發送時間。
```

## 6 種跟進情境 Prompts

### 📝 情境 1：會議後跟進

**使用時機**：剛開完會，當天或隔天發送
**目的**：感謝 + 重點整理 + 推進下一步

#### Prompt

```
請幫我寫一封「會議後跟進」訊息。

客戶資訊：
- 客戶姓名：[填入]
- 公司名稱：[填入]
- 會議日期：[填入]
- 會議主題：[填入]

會議重點：
1. [討論重點 1]
2. [討論重點 2]
3. [討論重點 3]

客戶提出的問題/疑慮：
- [問題 1]
- [問題 2]

約定的下一步：
- [下一步行動]

發送管道：[Email / LINE / WhatsApp]

請產出 3 個版本的跟進訊息，包含：
1. 感謝對方的時間
2. 會議重點摘要
3. 回應他們的問題/疑慮
4. 確認下一步行動
5. 明確的 CTA
```

#### 範例輸出

**版本 A：正式版**
```
王經理您好，

感謝您今天撥冗與我會面，討論貴公司的 AI 導入計畫。

以下是今天會議的重點整理：
1. 貴公司希望在 Q2 前完成第一階段導入
2. 主要目標是提升業務團隊的效率
3. 預算範圍約在 XX 萬

關於您提到的培訓時數問題，我已確認可以彈性調整為 2 天 + 1 天複習課的模式。

如您方便，我將在本週五前提供正式報價，届時再請您過目。

祝 商祺

[你的名字]
```

**版本 B：親切版**
```
王經理，

今天聊得很開心！感謝你花時間跟我分享貴公司的規劃 😊

快速整理一下今天的重點：
✓ Q2 前完成第一階段
✓ 重點是提升業務效率
✓ 培訓可以拆成 2+1 天

報價我週五前給你，有任何問題隨時敲我！

[你的名字]
```

**版本 C：簡短版**
```
王經理，

感謝今天的會議！

重點確認：Q2 導入、2+1 天培訓、週五前給您報價。

有問題隨時聯繫 👍

[你的名字]
```

### 💤 情境 2：沉默客戶喚醒

**使用時機**：客戶 1-3 個月沒聯絡
**目的**：重新建立連結，不要太銷售

#### Prompt

```
請幫我寫一封「沉默客戶喚醒」訊息。

客戶資訊：
- 客戶姓名：[填入]
- 公司名稱：[填入]
- 上次聯絡：[填入日期]
- 上次聊的內容：[填入]
- 當時沒成交的原因：[填入，如：預算/時機/需要考慮]

我想用的切入點：
□ 分享一個對他有用的資訊
□ 提到產業趨勢或新聞
□ 單純問候關心
□ 提到我們有新的服務/優惠
□ 其他：[填入]

發送管道：[Email / LINE / WhatsApp]

請產出 3 個版本，原則：
1. 不要太銷售感
2. 提供價值或關心
3. 輕鬆自然，像朋友問候
4. 有一個軟性的 CTA（不是「要不要買」）
```

#### 範例輸出

**版本 A：分享價值型**
```
李總，好久不見！

最近看到一篇關於製造業 AI 應用的報告，想到你之前提過的生產線優化，覺得裡面有些數據你可能會感興趣。

有空的話我可以轉給你參考，或是我們約個咖啡聊聊近況？

[你的名字]
```

**版本 B：關心問候型**
```
李總，

突然想到你，想說問候一下！

最近公司還好嗎？上次聊到的那個專案後來怎麼樣了？

有機會的話再聚聚 ☕

[你的名字]
```

**版本 C：新消息型**
```
李總，

我們最近推出了一個新的服務，專門針對像貴公司這樣規模的企業設計的，比之前的方案更彈性。

如果你最近有空，我可以花 15 分鐘跟你分享一下，看看對你們有沒有幫助？

[你的名字]
```

### 🎊 情境 3：節日問候

**使用時機**：新年、中秋、聖誕、農曆年等
**目的**：維繫關係，讓客戶記得你

#### Prompt

```
請幫我寫一封「節日問候」訊息。

客戶資訊：
- 客戶姓名：[填入]
- 關係程度：[陌生/普通/熟悉/很熟]
- 上次互動：[填入]

節日：[填入節日名稱]

我想傳達的感覺：
□ 正式祝福
□ 溫馨關懷
□ 輕鬆幽默
□ 簡單俐落

是否要提到業務？
□ 完全不提（純問候）
□ 輕輕帶過（年後有空聊聊）
□ 明確提到（新年新方案）

發送管道：[Email / LINE / WhatsApp]

請產出 3 個版本，原則：
1. 真誠，不要太制式
2. 符合節日氛圍
3. 根據關係程度調整語氣
```

#### 範例輸出（農曆新年）

**版本 A：正式版**
```
張總，

新年快樂！🧧

感謝過去一年的支持與信任，祝您新的一年：
事業順利、身體健康、闔家平安！

蛇年行大運！

[你的名字]
```

**版本 B：親切版**
```
張總～新年快樂！🎉

過年有沒有回老家？
祝你蛇年一切順利，紅包拿到手軟！

年後有空再約吃飯聊聊～

[你的名字]
```

**版本 C：簡短版**
```
張總，新年快樂！🧧
祝蛇年順利、平安健康！

[你的名字]
```

### 🎂 情境 4：生日祝福

**使用時機**：客戶生日當天或前一天
**目的**：個人化關懷，建立情感連結

#### Prompt

```
請幫我寫一封「生日祝福」訊息。

客戶資訊：
- 客戶姓名：[填入]
- 關係程度：[陌生/普通/熟悉/很熟]
- 我知道他的興趣/特點：[填入，如：愛喝咖啡/喜歡高爾夫/有小孩]

我想傳達的感覺：
□ 正式祝福
□ 溫馨個人化
□ 輕鬆幽默

發送管道：[Email / LINE / WhatsApp]

請產出 3 個版本，原則：
1. 個人化（不要像群發）
2. 如果知道他的興趣，融入祝福
3. 真誠簡短
4. 不要提業務
```

#### 範例輸出

**版本 A：正式版**
```
陳總，

生日快樂！🎂

祝您新的一歲：
身體健康、事業順利、心想事成！

[你的名字]
```

```
今天記得給自己放個假，來杯好咖啡慶祝一下！☕

祝新的一歲一切順心～
```

**版本 C：幽默版**

```
陳總！

聽說今天是某人的生日？🤔

祝你永遠 18 歲（的心態）！
```

### 📚 情境 5：分享價值

**使用時機**：看到對客戶有用的資訊
**目的**：提供價值，維持存在感，建立專業形象

#### Prompt

```
請幫我寫一封「分享價值」訊息。

客戶資訊：
- 客戶姓名：[填入]
- 產業：[填入]
- 他關心的議題：[填入]

我要分享的內容：
- 類型：[文章/報告/新聞/工具/活動]
- 標題或主題：[填入]
- 為什麼對他有用：[填入]
- 連結：[填入]

發送管道：[Email / LINE / WhatsApp]

請產出 3 個版本，原則：
1. 說明為什麼想到他
2. 簡述重點（不用看完整篇也知道在講什麼）
3. 輕鬆的 CTA（有興趣可以看看）
4. 不要銷售感
```

#### 範例輸出

**版本 A：專業版**
```
林經理，

剛看到這篇關於「2026 製造業 AI 應用趨勢」的報告，裡面提到的幾個數據蠻有意思的，想到你之前提過想了解同業怎麼做，分享給你參考。

重點摘要：
• 67% 的製造業已開始導入 AI
• ROI 最高的應用是品質檢測
• 導入失敗的主因是缺乏內部訓練

完整報告：[連結]

有什麼想法歡迎交流！

[你的名字]
```

**版本 B：輕鬆版**
```
林經理～

看到這篇文章想到你！

講的是製造業 AI 的趨勢，有些數據蠻值得看的，特別是那個 67% 的導入率... 你們同行動作都很快欸 😅

有空可以看看：[連結]

[你的名字]
```

**版本 C：簡短版**
```
林經理，

這篇分享給你，講製造業 AI 趨勢的，跟你之前問的很相關 👇

[連結]

覺得有意思可以聊聊！

[你的名字]
```

### 🤝 情境 6：轉介紹請求

**使用時機**：客戶合作順利、滿意度高時
**目的**：請客戶幫忙介紹，擴大業務範圍

#### Prompt

```
請幫我寫一封「轉介紹請求」訊息。

客戶資訊：
- 客戶姓名：[填入]
- 合作多久：[填入]
- 合作成果：[填入具體成果]
- 關係程度：[普通/熟悉/很熟]

我希望他介紹的對象：
- 產業：[填入]
- 職位：[填入]
- 或描述：[填入]

我有提供轉介紹獎勵嗎？
□ 沒有
□ 有，內容是：[填入]

發送管道：[Email / LINE / WhatsApp]

請產出 3 個版本，原則：
1. 先肯定合作成果
2. 說明希望服務更多像他這樣的客戶
3. 具體說明希望介紹什麼樣的人
4. 讓對方好開口（提供介紹的說詞）
5. 不要有壓力感
```

#### 範例輸出

**版本 A：正式版**
```
吳總，

首先感謝這半年來的信任，很高興看到培訓後團隊的效率提升了 30%！

我們一直希望能服務更多像貴公司這樣重視團隊成長的企業。如果您身邊有朋友或同業也在考慮 AI 培訓，不知道方不方便幫我引薦？

我可以先提供免費的 30 分鐘諮詢，讓他們評估是否適合。

當然，如果現在不方便也完全沒關係，只是想到就問問。

再次感謝您的支持！

[你的名字]
```

**版本 B：輕鬆版**
```
吳總～

上次看到你們團隊的成果報告，30% 的效率提升真的很棒！🎉

對了，如果你有朋友也在煩惱團隊 AI 技能的問題，歡迎介紹給我！我可以先免費聊聊，看看能不能幫上忙。

你可以直接跟他說：「我有個朋友專門做企業 AI 培訓，效果不錯，要不要聊聊？」

沒有的話也沒關係啦，就順口問問 😄

[你的名字]
```

**版本 C：簡短版**
```
吳總，

最近有朋友問起 AI 培訓的事，想到你們的成果很不錯。

如果你身邊也有人有需要，歡迎介紹給我，我可以先免費諮詢。

感謝！🙏

[你的名字]
```

| 0:50-1:00 | 分享 | 展示成果、Q&A |
